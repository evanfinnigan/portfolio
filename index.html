Stack Attack (CS 4480: Game Development Project - UWO)

Language: C# (built in Unity)

Stack Attack is a local-multiplayer platform fighting game, in which players can choose to form & break alliances on the fly. Players stack on top of each other to gain stronger abilities against the remaining opponents, and the last stack standing wins. This was a group project built for the Game Development Project course at UWO. The implementation of this game was a group project, though I created the game design for a solo project.

Some cool features of this project include:

  • Local multiplayer
  • Player vs. AI gameplay
  • Unique alliance mechanics
  • Different abilities: melee combat and two types of projectile combat

Vikings (CS 4483: Game Design - UWO)

Language: C# (built in Unity)

Vikings! is a retro-style game in which the player explores the seas in a Viking ship, battles sea monsters, and pillages towns for supplies and crew. The implementation of this game was a group project, though I created the game design for a solo project.
