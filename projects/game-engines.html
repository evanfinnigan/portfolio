Aerial Firefighting

Web Demo

View the Project on Github


This project was created for CS 4482: Game Engines at The University of Western Ontario.

In Aerial Firefighting, you fly a rescue plane over a procedurally-generated island, and you must put out forest fires in order to save the forests and towns on the island. Time is limited, as the fire spreads quickly and you have a limited water supply. If you run out of water, you'll have to go off-route, flying down over the ocean to gather more.

This was an interesting and fun project for me. Some cool features of the game include procedural terrain generation and a dynamic skybox and lighting depending on the time of day.
