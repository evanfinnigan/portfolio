Graph Algorithms & AI



An introductory AI project

During this project, I implemented the following algorithms in python:

  • Depth First Search
  • Breadth First Search
  • Uniform Cost Search
  • A* Search
  • Minimax with Alpha/Beta Pruning

This project famously originates from CS188 at UC Berkely.

I did this project a number of years ago, so I am currently trying to locate the project files. They may be lost to the sands of time. If I locate them, I may post a post more about it.
